类的对象只有有限多个 称之为枚举类
如果枚举类只有一个对象 则可以用作单例模式实现

# 自定义枚举类

```java
class Season {
    private final String seasonName;
    private final String seasonDesc;

    private Season(String seasonName, String seasonDesc) {
        this.seasonDesc = seasonDesc;
        this.seasonName = seasonName;
    }

    public static final Season SPRING = new Season("春天", "1");
    public static final Season SUMMER = new Season("夏天", "2");
    public static final Season AUTUMN = new Season("秋天", "3");
    public static final Season WINTER = new Season("冬天", "4");

    public String getSeasonDesc() {
        return seasonDesc;
    }

    public String getSeasonName() {
        return seasonName;
    }

    @Override
    public String toString() {
        return "Season{" +
                "seasonName='" + seasonName + '\'' +
                ", seasonDesc='" + seasonDesc + '\'' +
                '}';
    }

```



# enum枚举类

```java
enum Season1 {
        // 提供当前的枚举类的对象
        SPRING("春天", "1"),
        SUMMER("夏天", "2"),
        AUTUMN("秋天", "3"),
        WINTER("冬天", "4"),;
        private final String seasonName;
        private final String seasonDesc;

        public String getSeasonName() {
            return seasonName;
        }

        public String getSeasonDesc() {
            return seasonDesc;
        }

         Season1(String seasonName, String seasonDesc) {
            this.seasonDesc = seasonDesc;
            this.seasonName = seasonName;
        }
    }
```

> 必要时候可以自行重写toString方法



## 常用方法

- values
- valueOf（String str）
- toString

```java
// 获取枚举类的对象
        Season1[] s = Season1.values();
        for (Season1 season1 : s) {
            System.out.println(season1);
        }
        // valueOf 没有找到抛出异常
        Season1 winter = Season1.valueOf("WINTER");
        System.out.println(winter);
    }
```



# enum枚举类实现接口

```java
    interface show {
        public void show();
    }
    class Sea implements show {

        @Override
        public void show() {
            // 如果每个枚举类都需要都有自己不同的重写方法的实现 则注释本地
          	// 对每个枚举对象进行重写show方法
            System.out.println("实现接口");
        }
    }
```
